# Agent Pet Companion Logo Assets / 品牌图标资产

This directory contains the approved Agent Pet Companion app icon and reusable standalone brand mark.

此目录保存已确认采用的 Agent Pet Companion App 图标与可复用的透明品牌主体。

## Files / 文件

- `agent-pet-app-icon-master-1024.png`: approved 1024 px app icon master / 已确认的 1024 px App 图标母版。
- `png/`: common square PNG sizes from 16 px through 1024 px / 16–1024 px 常用方形 PNG。
- `transparent/agent-pet-mark-transparent-1024.png`: standalone irregular mark with an alpha channel / 仅含主体、带 Alpha 透明通道的不规则图标。
- `macos/AgentPetCompanion.iconset/`: standard macOS 1x and 2x icon source set / macOS 标准 1x、2x 图标源文件。
- `macos/AgentPetCompanion.icns`: packaged macOS icon resource / macOS 图标资源包。
- `macos/AgentPetCompanionTransparent.iconset/`: transparent 1x and 2x app icon source set / 透明版 1x、2x App 图标源文件。
- `macos/AgentPetCompanionTransparent.icns`: transparent app icon used by the macOS bundle / macOS 安装包实际使用的透明版 App 图标。

## Design intent / 设计含义

The loop represents a continuously active Agent signal; the inner lights give it a companion-like presence; the floating spark represents response and inspiration. The mark intentionally avoids paw prints and literal animal silhouettes.

环形代表持续运行的 Agent 信号，内部光点赋予其伙伴般的生命感，上方火花代表回应与灵感。设计刻意避开爪印和具体动物轮廓。

## Generation provenance / 生成来源

The approved icon was produced with the built-in image generation workflow. The transparent mark was derived from the approved icon using a flat chroma-key edit followed by local alpha extraction.

已确认图标由内置图像生成流程制作；透明主体以该图标为参考生成纯色键控版本，再在本地提取 Alpha 通道。
